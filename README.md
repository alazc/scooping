# Bimanual Granular Scooping

Project website for a dual-armed mobile robot that perceives objects through RGB-D sensing, plans grasps in 3D, navigates autonomously, and scoops granular materials using coordinated two-arm control.

**Live site:** [alazc.github.io/scooping](https://alazc.github.io/scooping/)
