Task 1

Manipulation

The first task focuses on bimanual object manipulation using AnyGrasp for grasp pose generation and SAM3 for real-time object tracking. The system processes RGB-D input to detect grasp candidates, then uses segmentation masks with PID-based tracking to coordinate dual-arm pick-and-place of rigid objects.

Task 2

Manipulation & Locomotion

The second task adds a navigation action server to the Task 1 pipeline. The robot uses PID control for straight-line path following with initial and ending turns to approach target objects. It shares the same AnyGrasp + SAM3 methodology as Task 1, with the addition of coordinated base locomotion during manipulation.

Task 3

Scooping

The final task combines everything into a candy pickup pipeline: the robot picks up a scoop and bucket, moves to the candy box, scoops candy, and drops it in the bucket. This integrates bimanual grasping, navigation, and coordinated tool use into a single end-to-end sequence.

Key Components

  • ArUco detection node — localizes scoop and bucket via fiducial markers
  • Execute grasp server — coordinates bimanual grasp sequences
  • Waypoint trajectories — pre-planned arm paths for scooping and transfer motions
